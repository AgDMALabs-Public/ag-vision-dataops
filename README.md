# ag_vision
* The main objectives of this package is to help with data ops on data used in vision ML projects in Ag.
* That will include things like:
1. Ingesting data into a standard format to databricks.
2. Propagating images with metadata in a standrd way.
3. Connection and helper functions to get data to and from databricks.
4. Pipelines to generate tables indexing all the data and the metadata about that info.
5. Pipelines to run simple ML models on the images to enrich them
6. Pipelines to process drone imagery.
